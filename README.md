# RoboCon Timer 2026

ロボット競技（ロボコン）向けに特化した、遅延ゼロのリアルタイムタイマー＆スコアボードシステムです。
操作用のコントロールパネル、会場用のプロジェクター画面、および配信用（OBS等向け）のクロマキー対応オーバーレイ表示の3種類の画面を、**1つのPythonサーバーをベースに完全同期**して管理します。

---

## 🌟 主な機能・特徴

1. **用途に合わせた3つの独立画面**
   - **操作メイン画面 (`index.html`)**: 大会運営が手元で操作するためのフル機能コントロールパネル。スマホでのレスポンシブ操作にも対応しています。
   - **プロジェクター画面 (`screen.html`)**: 会場のスクリーンに映し出すための、タイマーとスコア表示に特化したフルスクリーン用グラフィック。
   - **透過合成用画面 (`overlay.html`)**: OBS Studioなどの動画配信ソフトで、カメラ映像に合成するための画面（緑背景/クロマキー対応）。文字は強いシャドウで縁取られており、暗い場所でも白い機体の前でも潰れません。

2. **超低遅延（60fps相当）同期**
   サーバー・クライアント間の非同期通信を最適化し、ブラウザの処理落ちや通信ラグによるタイマーの「ズレ」が一切発生しない15msごとのポーリング式状態同期を実装しています。

3. **自動機 / 手動機の高度なスコア設計**
   同一のアクションでも、手動制御と自動制御で得点が異なる仕様に対応。画面上の「自動機モード」チェックを切り替えるだけで得点配分を柔軟に変動させ、画面上に手動/自動それぞれの「カウント内訳」を詳細に表示します。

4. **キーボードショートカット対応**
   マウスやタップだけでなく、事前に割り当てられたキーボードのキーを押すだけで即座に得点を加算（または `Ctrl + キー` で減算）できます。各項目の横に `[Q]` のように操作キーがガイド表示されます。

5. **音響と演出のエフェクト（Web Audio API）**
   残り3秒からのカウントダウン音（500Hz）と、ゼロ時点でのブザー音（1000Hz）をブラウザ本体が生合成で出力。同時に、残り3秒からタイマーの文字色が黄色く発光・点滅（フラッシュ）し、会場や配信映像越しに焦燥感と緊張感を演出します。

## 🚀 起動方法

このシステムはローカル上で動作する軽量な Python Web サーバーを必要とします。

1. **サーバーの起動**
   コマンドプロンプトまたはターミナルを開き、本フォルダ内で以下のコマンドを実行します。
   ```bash
   python server.py
   ```
   > これにより、`localhost:8080` でサーバーが待機状態になります。

2. **各種画面へのアクセス**
   目的に応じてブラウザやOBSのブラウザソースから以下のURLを開いてください。

   - 🕹️ **操作・管理用画面**
     `http://localhost:8080/`
     *(※手元のPCまたは、同じWi-Fiに繋がったスマホからサーバPCのIP `http://192.168.x.x:8080/` を開いて操作可能)*

   - 📽️ **会場プロジェクター用**
     `http://localhost:8080/screen.html` (F11キー等で全画面化して利用)

   - 🎥 **OBS 配信合成用**
     `http://localhost:8080/overlay.html`
     *(※OBSのフィルタ設定にて「クロマキー：緑色」を適用してください)*

## ⚙️ 各種設定ファイル

大会のルールやチーム設定を変更したい場合は、`data/` フォルダ内のJSONファイルを書き換えるだけでリアルタイムに操作パネルが組み替わります。

### 1. チーム名リスト設定 (`data/teams.json`)
参加するチームの名称を配列形式で記述します。
```json
[
    "TEAM ALFA",
    "TEAM BRAVO",
    "TEAM CHARLIE"
]
```

### 2. 得点ルールとキーバインド設定 (`data/scores.json`)
競技課題（タスク）ごとに、点数、挙動タイプ、バインドするキーを設定します。

```json
[
    {
        "id": "ring",
        "label": "リング",
        "points": {
            "manual": 10,
            "auto": 20
        },
        "type": "number",
        "keyRed": "a",
        "keyBlue": "l"
    },
    {
        "id": "vgoal",
        "label": "Vゴール",
        "points": {
            "manual": 0,
            "auto": 0
        },
        "type": "toggle",
        "keyRed": "e",
        "keyBlue": "o"
    }
]
```

#### パラメータ解説
- `points`: 手動機と自動機での得点が異なる場合は `{ "manual": 10, "auto": 20 }` のように設定。共通にしたい場合は単純に `10` と単独で数値を書いても下位互換で動きます。
- `type`: 回数をカウントする場合は `"number"`、一撃で試合終了させたいスイッチ類（Vゴール等）は `"toggle"` を指定します。
- `keyRed` / `keyBlue`: 加算操作を紐付けるキーボードのキーです。Ctrlキーと同時に押すと「減点」として機能します。

## 📱 スマートフォンからの操作について
操作メイン画面は「レスポンシブデザイン」を採用しています。
画面幅の細いスマートフォンで開いた場合は、誤タップを防ぐための大きな（50px四方）入力ボタンへと変形し、左右のチームパネルが上下に並ぶ縦画面操作仕様へと自動的にトランスフォームします。運営者がアリーナの任意の場所から直感的に操作することが可能です。
